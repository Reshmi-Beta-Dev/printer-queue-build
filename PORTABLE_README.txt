Print Queue Monitor - Portable Version
=====================================

INSTALLATION:
1. Run INSTALL.bat as Administrator
2. The service will be installed and started automatically

UNINSTALLATION:
1. Run UNINSTALL.bat as Administrator

CONSOLE MODE:
1. Run run-console.bat to test in console mode

CONFIGURATION:
Edit appsettings.json to customize settings

OUTPUT:
Print jobs are saved to: C:\PrintQueueJobs
